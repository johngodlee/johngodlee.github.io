R code for calculating woody productivity and vital rates

2023-03-17

I have developed some R code to use with data from the SEOSAW database to calculate woody productivity and vital rates according to Talbot et al. (2014), Kohyama et al. (2018) and Kohyama et al. (2019). I’ve archived the code as a Github repository in case anybody wants to use it.
