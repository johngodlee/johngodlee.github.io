LaTeX plot census forms

2021-02-25

In addition to the ODK forms I created to help collaborators collect woodland plot data for the SEOSAW plot network, I wanted to make sure that we could guide data collectors in collecting SEOSAW-style data even if they don’t have access to an Android device.

I used LaTeX to create some forms which prompt users on the types of data to collect to describe a new plot. The idea is that the form is used in conjunction with the SEOSAW field manual.

Here is a copy of the form
